# Clueless Commerce

Landing site for **cluelesscommerce.com**, inspired by [boco.agency](https://boco.agency): design-led e-commerce agency layout with similar animations (infinite marquees, scroll-triggered reveals).

## Stack

- **Vite** + **React** + **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (scroll and entrance animations)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Sections

- **Hero** – Tagline + infinite horizontal marquee(s) of store screenshots
- **About** – Mission and “View all Projects” CTA
- **Projects** – Horizontal marquee of project cards (CRO/ATC stats)
- **Partners / Build stack** – Marquee of tech logos
- **Services** – Three columns (Research & Design, Web Dev, CRO)
- **Testimonials** – Carousel with animated transitions
- **Careers CTA** – “Join us” block
- **Contact** – “Book a call”
- **Footer** – Tagline, nav, social, copyright

Replace placeholder project names, testimonials, and “Book a call” URL with real content and assets when ready.
